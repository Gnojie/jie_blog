## 重绘与回流

看一下你的简历，是不是都在陈述业务的支撑数量？——
 A 系统配合业务方做了 1、2、3、4 的功能；
 B 业务配合合作方做了 1、2、3 的功能；
 C 功能配合做了 XXX 的模块
  ... 
没有 PM 的总控能力体现、
没有主导角色的技术建设落地、
没有成长性的总结沉淀布道，
